# Overleaf Upload Notes

This folder is ready to upload to Overleaf.

## Easiest workflow
1. Go to [https://www.overleaf.com](https://www.overleaf.com)
2. Log in
3. Click `New Project`
4. Click `Upload Project`
5. Upload this folder as a zip file, or upload all files inside this folder
